# Blackboard Aptana Theme for Atom.io

A *modified* adaptation of the Blackboard Aptana theme for Atom inspired by [Blackboard Theme](https://atom.io/themes/blackboard-theme)

### Supported Specific Languages

* HTML
* Javascript
* PHP
* Python
* Typescript
